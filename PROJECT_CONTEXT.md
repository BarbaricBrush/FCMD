# FCMD Project Context

Purpose
- Static marketing site for FCMD.

Stack
- HTML/CSS/JS only.
- Assets in `images/`.
- Pages: `index.html` (home), `experience.html` (secondary page).

Key Files
- `index.html`
- `experience.html`
- `styles.css`
- `script.js`
- `images/`

Local Preview
- Open `index.html` in a browser, or
- Run `python3 -m http.server` and visit `http://localhost:8000`.

Deployment
- GitHub Pages: set Pages to branch `main` and root `/`.
- Custom domain: set CNAME in GitHub Pages + DNS CNAME to `yourname.github.io`.

Notes
- Avoid committing OS files like `.DS_Store`.
